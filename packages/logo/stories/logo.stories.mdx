# Logo

  [![](https://img.shields.io/npm/v/@digdir/ds-logo?label=@digdir/ds-logo&color=0051be)](https://www.npmjs.com/package/@digdir/ds-logo)

På denne siden finner du instrukter til hvordan du kan ta i bruk logo pakken til Digdir.

## Kom i gang

### NPM

  [![](https://img.shields.io/npm/dm/@digdir/ds-logo.svg)](https://www.npmjs.com/package/@digdir/ds-logo)

Bruk av logoene gjennom NPM (Node Package Manager).

### 1. Installer pakken i kommandolinjen ved hjelp av NPM eller Yarn:

  #### NPM
  `npm install @digdir/ds-logo`
#### Yarn
  `yarn add @digdir/ds-logo`

### 2. Importer logoene inn på denne måten:

#### React
<CodeSnippet children='
import { DigdirLogo } from "@digdir/ds-logo"
' language='javascript'/>


  #### JavaScript

<CodeSnippet children='
import DigdirLogo from "@digdir/ds-logo/svg/logo_digdir.svg"
' language='javascript'/>
