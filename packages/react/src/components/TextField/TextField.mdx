# TextField

<Information text='beta' />

`TextField` er en tekstboks som tar imot korte, rene tekster uten linjeskift. For lengre tekster, bruk `TextArea`.

## Bruk

```tsx
import { TextField } from '@digdir/design-system-react';

<TextField {...props} />;
```
