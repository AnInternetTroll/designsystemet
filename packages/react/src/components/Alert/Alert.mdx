# Alert

`Alert` skal brukes til å vise viktige meldinger, varsler eller advarsler til brukere. Den er designet for å fange brukernes oppmerksomhet og gi dem relevant informasjon som krever handling eller gjøre dem bevisst på viktig informasjon

## Tilgjengelighet

### Alternativ-tekst

Ikonene er ikke bare dekorative. De har default alternativ-tekst som kommuniserer alvorlighetsgrad: "Informasjon", "Suksess", "Advarsel" og "Feil". Denne blir lest opp som en del av innholdet for skjermleserbrukere.

### Interaksjon Skjermleser

Alert-komponenten er ikke en ARIA live region og er ikke relatert til ARIA `role="alert"`. Komponenten skal ikke motta dynamiske oppdateringer. Den blir presentert for skjermleserbrukere som vanlig statisk innhold.

Du kan legge på `role=alert` men vær opperksom på forventet oppførsel dette medfører. Les mer om dette på [W3 Alert pattern](https://www.w3.org/WAI/ARIA/apg/patterns/alert/)
