# Kjernekomponenter
Kjernekompoenter til bruk i Digdir.
## Kom i gang

[![](https://img.shields.io/npm/v/@digdir/design-system-react?label=@digdir/design-system-react&color=0051be)](https://www.npmjs.com/package/@digdir/design-system-react)
[![](https://img.shields.io/npm/dm/@digdir/design-system-react.svg)](https://www.npmjs.com/package/@digdir/design-system-react)
[![](https://img.shields.io/badge/licence-BSD3-yellowgreen?color=yellowgreen)](https://opensource.org/licenses/BSD-3-Clause)

Bruk av kjernekomponenter gjennom NPM (Node Package Manager).

### 1. Installer pakken i kommandolinjen ved hjelp av NPM eller Yarn:

<CodeSnippet children='npm install @digdir/design-system-react' language='markup'/>
<CodeSnippet children='yarn add @digdir/design-system-react' language='markup'/>

### 2. Importer komponentene inn og ta dem i bruk på denne måten:

<CodeSnippet children='// Importer
import { Button } from "@digdir/design-system-react"
// Bruk komponenten
<Button variant="secondary">Les mer</Button>' language='javascript'/>
