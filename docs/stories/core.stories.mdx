### 1. Installer pakken i kommandolinjen ved hjelp av NPM eller Yarn:

#### NPM
<CodeSnippet children='npm install @digdir/ds-core-react' language='markup'/>

#### Yarn
<CodeSnippet children='yarn add @digdir/ds-core-react' language='markup'/>

#### NPM
`npm install @digdir/ds-tokens styled-components`
#### Yarn
`yarn add @digdir/ds-tokens styled-components`
